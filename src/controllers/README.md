# Dossier `controllers/` — Contrôleurs

Ce dossier contient tous les contrôleurs de l'API. Chaque contrôleur gère la logique métier pour un domaine fonctionnel spécifique.

## Fichiers

| Fichier | Rôle |
|---|---|
| `adminContentController.js` | Gestion administrative des contenus (création, modification, approbation, suppression) |
| `adminPhotoController.js` | Gestion administrative des photos (téléchargement, modification, approbation, suppression) |
| `adminPlaylistController.js` | Gestion administrative des listes de lecture |
| `adminUserController.js` | Gestion administrative des utilisateurs (listing, modification, suppression, gestion des quotas) |
| `aiController.js` | Interactions avec le service d'intelligence artificielle (chat) |
| `authController.js` | Authentification des utilisateurs (inscription, connexion, déconnexion, mot de passe, profil) |
| `cartController.js` | Gestion du panier d'achat (ajout, suppression, consultation) |
| `checkoutController.js` | Gestion du processus de paiement et de validation de commande |
| `contentController.js` | Consultation publique des contenus (liste, détails) |
| `dashboardController.js` | Statistiques et données du tableau de bord administratif |
| `favoriteController.js` | Gestion des favoris des utilisateurs |
| `inquiryController.js` | Gestion des demandes de renseignements (création, listing, réponse) |
| `mediaController.js` | Diffusion des fichiers média (streaming vidéo, téléchargement, images) |
| `notificationController.js` | Gestion des notifications et des abonnements push |
| `packController.js` | Gestion des packs d'abonnement (création, modification, suppression, listing) |
| `paymentController.js` | Gestion des paiements (création, vérification, webhooks, simulation) |
| `photoController.js` | Consultation publique des photos (liste, détails, téléchargement) |
| `playlistController.js` | Consultation publique des listes de lecture |
| `searchController.js` | Recherche globale dans les contenus et les photos |
