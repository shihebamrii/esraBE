# Dossier `scripts/` — Scripts Utilitaires

Ce dossier contient des scripts exécutables en ligne de commande pour des tâches de maintenance, de test et d'initialisation.

## Fichiers

| Fichier | Rôle |
|---|---|
| `check-models.js` | Vérifie que les modèles Mongoose se chargent correctement et peuvent se connecter à la base de données |
| `list-all-models.js` | Liste tous les modèles Mongoose enregistrés dans l'application |
| `seed-users.js` | Crée des utilisateurs de test (admin et user) pour initialiser la base de données |
| `test-ai-service.js` | Teste le fonctionnement du service d'intelligence artificielle (Google Gemini) |
