# Dossier `config/` — Configuration

Ce dossier contient les fichiers de configuration de l'application.

## Fichiers

| Fichier | Rôle |
|---|---|
| `database.js` | Gère la connexion à la base de données MongoDB avec Mongoose |
| `index.js` | Centralise toutes les variables de configuration (serveur, JWT, base de données, paiement, stockage, etc.) à partir des variables d'environnement |
